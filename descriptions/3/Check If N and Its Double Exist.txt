Учитывая массив arr целых чисел, проверьте, существуют ли два индекса i и j такие, что:

i != j
0 <= i, j < arr.length
arr[i] == 2 * arr[j]
 

Пример 1:
    Ввод: arr = [10,2,5,3]
    Вывод: true
    Объяснение: Для i = 0 и j = 2, arr[i] == 10 == 2 * 5 == 2 * arr[j]

Пример 2:
    Ввод: обр = [3,1,7,11]
    Вывод: ложь
    Объяснение: нет i и j, удовлетворяющих условиям.


Ограничения:

2 <= длина обр. <= 500
-103 <= обр[i] <= 103
