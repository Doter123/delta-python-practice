Напишите функцию, которая находит самую длинную строку общего префикса среди массива строк.

Если общего префикса нет, вернуть пустую строку "".

 

Пример 1:

Ввод: strs = ["цветок","поток","полет"]
Вывод: "фл"
Пример 2:

Ввод: strs = ["собака","гоночная машина","машина"]
Выход: ""
Объяснение: Среди входных строк нет общего префикса.
 

Ограничения:

1 <= длина строки <= 200
0 <= строки[i].length <= 200
strs[i] состоит только из строчных латинских букв.
